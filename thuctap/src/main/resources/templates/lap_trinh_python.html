Lý do bạn nên học lập trình Python ?

Ngôn ngữ lập trình Python vừa là ngôn ngữ hướng thủ tục và cũng là ngôn ngữ lập trình hướng đối tượng.

Tốc độ thực thi nhanh, thư viện STL với các thuật toán và cấu trúc dữ liệu được cài đặt sẵn là lí do Python là lựa chọn số một trong các cuộc thi về lập trình.
